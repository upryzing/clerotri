# User badges

Badges are small icons that appear on your profile. There are two main groups of badges - platform-wide badges, and Clerotri badges.

## Platform-wide badges

Platform-wide badges are given out by the Upryzing/Stoat instance you're using. **These badges appear on most apps.**

You can find a list of badges on the official Stoat instance (`stoat.chat`) [on their support site](https://support.stoat.chat/kb/account/badges).

## Clerotri badges

On top of the platform-wide badges, there are some Clerotri-specific badges. **These badges will only appear in Clerotri.**

### Built-in

If you're using the official Stoat instance (`stoat.chat`), these will always appear.

| Name               | Description                                                       | Obtainable |
| ------------------ | ----------------------------------------------------------------- | ---------- |
| Clerotri Developer | Main developers of Clerotri (TaiAurori and Rexogamer)             | ❌         |
| raccoon            | Insert, one of the founders of Stoat                              | ❌         |
| Lea's Paw          | Lea, former Stoat team member and the original creator of AutoMod | ❌         |

### Extra badges

These badges will only appear if you enable extra badges in the app, and are fetched from Clerotri's servers. You can find out more about data handling for these in [the privacy document](./privacy.md#extra-badges).

#### Donor badges

These badges are given to those who have donated various amounts of money to Clerotri's developers.

If you want a donor badge, [**donate via Ko-fi or Liberapay**](https://rexowo.dev/donate?utm_source=clerotri_badge_docs). Feel free to ask in [the Clerotri server](https://stt.gg/clerotri) if you want to donate via another method.

After donating, [email `clerotri@upryzing.app`](mailto:beartechtalks@gmail.com?subject=Donor+badge+request) with proof of your donation and your user ID. You can get your user ID by opening the settings menu and selecting "Copy Debug Info" - **you only need to include the `userID` field**.

The badge tier will be based on **the amount of money you've given so far**. Bear this in mind if you're donating via Liberapay - for example, if you pick the Medium tier, you will be able to claim the Silver badge if you fund it for a month and a half.

The following amounts are in **British pounds**.

| Name     | Amount |
| -------- | ------ |
| Bronze   | £1     |
| Silver   | £5     |
| Gold     | £10    |
| Platinum | £25    |
| Diamond  | £50    |
